Suma de dos números


El algoritmo de suma de dos números es uno de los ejercicios fundamentales en el aprendizaje de la programación, ya que introduce el uso de variables, entrada de datos y operaciones aritméticas básicas. A través de este ejemplo, el lector podrá comprender cómo se capturan datos, se procesan y se presenta un resultado.

Este tipo de algoritmo permite reforzar la estructura general de un programa, basada en tres elementos principales: entrada, proceso y salida.

Estructura del algoritmo:
Entrada: Dos números ingresados por el usuario.
Proceso: Realizar la suma de ambos valores.
Salida: Mostrar el resultado de la suma.



En este algoritmo, se observa cómo los datos ingresados por el usuario son almacenados en variables, posteriormente procesados mediante una operación aritmética y finalmente presentados como resultado.

Este ejercicio es clave, ya que introduce el flujo completo de un programa y sienta las bases para el desarrollo de algoritmos más complejos, donde se combinarán múltiples operaciones y estructuras.

En conclusión, la suma de dos números no solo representa una operación matemática simple, sino también un ejemplo esencial para comprender cómo funciona un programa desde la entrada de datos hasta la generación de resultados.


En este algoritmo se definen tres variables:
Numero1 como real.
Numero2 como real.
Suma como real.
El proceso será la suma de los dos números, numero 1 + numero2 (suma = numero1 + numero2)



El reto consiste en modificar el algoritmo de suma de dos números y en vez de sumar dos números van a ser tres números los que se van a utilizar, y mostrar el resultado.
Al trabajar con operaciones básicas, es recomendable utilizar nombres de variables claros y significativos como: num1, num2, suma o resultado. También es importante mantener una estructura limpia.
Uno se los errores más comunes es no convertir de forma correcta los datos ingresados por el usuario, especialmente en Python.
